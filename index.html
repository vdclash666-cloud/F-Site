Como fuciona o Head

Tudo que está dentro do Head será oque o site ira ler são chamados dem metadados Ex: autor, palavras chaves, padronização de caracteres, idioma, descrições, adaptação de tela

Como cada metadado fuciona

Os metas ou metadados são usados dentro do head com o nome de tag meta para definir oque o site lerá exemplos deles são:
Description: descreve oque o site insinua. Ex: aulas, estudos, etc
Keywords: palavras chaves que decidirão o entuito do seu site para pesquisas
Author: define o criador do site
Viewport: é oque define o tamanho da fonte para certos dispositivos

Como fuciona o Body

O body será tudo aquilo que o leitor verá no site incluido outras tag para dar mais sentido ao texto e algumas marcações.

Negrito: "b" Ex
Linha divisória: "hr" EX
Pular uma linha: "br" Ex
Deixar pequeno: "small" Ex
Riscado: "del" ou "strike" "strike" está em decaimento Ex
Texto na parte superior: "sup" Ex
Texto na parte inferior: "sub" Ex
Marcação (amarela): mark "Ex"
Sublinhado: "u"
itálico: "i"
citar nome de uma abreviação: "abbr" Ex: HTML
